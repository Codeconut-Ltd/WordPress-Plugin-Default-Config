Config plugin for enhanced security and frontend performance.
Generic and theme-independent implementation with a modern code style.


- - -


== Features ==

Disables some default core features to enhance security and frontend performace.


= Warning =

The settings in this theme are hardcoded and potentially breaking some advanced features in themes.
Enabling this plugin might also interfere with 3rd party application functionality (e.g. the WordPress mobiel app).


- - -


== How to use ==

= Git workflow =

Use of submodules is recommended:

    git submodule add USER:REPOSITORY wp-content/plugins/codeconutltd-global

Call this from web root. The path must not be preceded with a slash.
